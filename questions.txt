Чем, по-вашему, Python превосходит остальные языки программирования?
Для каких задач он подходит лучше всего? Для чего он не подходит?

Большое количество библиотек. Подходит для web разработки и анализа данных (ML).
Не подходит для разработки мобильных приложений.
С одной стороны, удобно, что не надо определять тип данных. С другой - динамическая типизация оборачивается
низкой скоростью работы программы из-за большого резерва памяти для каждой переменной.

Какие особенности Python вам доставляют больше всего неудобств?

Медленная скорость и GIL

Возможно ли писать программы на Python в функциональном стиле? Насколько он
для этого подходит?

Python не является функциональным языком программирования,
но его возможностей хватает, чтобы разрабатывать программы в функциональном стиле.
